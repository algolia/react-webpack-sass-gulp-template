# React + Webpack + SASS + Gulp template

This is a simple template for building React applications & components.

## Features

 * Powered with Gulp & Webpack
 * ES2015 (+ linter)
 * SASS (+ linter)
 * Minification & uglification
 * Hot reload
 * Deploy to gh-pages

## How to use it?

### Setup

```sh
$ npm install
# optional (HTTP & Live reload port configuration)
$ cp .env.example .env
```

### Development

```sh
$ npm run dev
```

### Other commands

```sh
# Clean
$ npm run clean

# Run linters
$ npm run lint

# Build release
$ npm run build

# Build + deploy to gh-pages
$ npm run deploy
```
